Website Layout Using Bootstrap

This is an example of a responsive webpage layout using BOOTSTRAP. Many buttons and links however are blank and are simply used as a demonstration.

Built with that stylesheet.

Basically, this is a website built with Bootstrap. Bootstrap is just a (very) large stylesheet with classes that have been designed by specialists. Knowing those classes makes responsive web-development easy. This is what I did here.

HTML5

We use the latest version of HTML, HTML5.

BOOTSTRAP

We use the latest version of Bootstrap, Bootstrap.

CSS

We use the latest version of CSS, CSS3.

If you want to use it...

First, you will need to import the Bootstrap file into your html files.

The Bootstrap file is the external stylesheet that we import using the "link" tag.

You will then also need to insert the same line of code into the head of your subpages, if you have any.

Resize the Window

The css styles change depending on the width of your browser window

For example, when viewing this website on a desktop, elements will be displayed in a row. Whereas, when viewing on a phone, those elements will be displayed vertically.

This ensures that everyhting is displayed nicely, and there are not elements that are too large, small, or too stretched out.
